## useful cmds for me

Package manager
```
:Mason
```
Removed cache configuration
```
rm -rf ~/.local/share/nvim
```
List of changes in file
```
:Git blame
```
left side tree of dictionaries - also you can use shortcut ctrl + n
```
:NvimTreeToggles
```
Place where is nvim repo
```
cd ~/.config/nvim
```
Show hidden files in nvim tree panel
```
H
```
